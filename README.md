Requiements
-----------

* PHP5.4+

Installation
------------

Modify your composer.json and run `php composer.phar update`

``` json
{
    "require": {
        "kzykhys/php-collection": "dev-master"
    }
}
```

Components
----------

* ArrayAccess
* Collection
* MutableCollection
* Tree
